# Scientific Calculator

A responsive and user-friendly **Scientific Calculator** built using HTML, CSS, and JavaScript. This project supports both basic and advanced mathematical calculations with a clean interface and smooth functionality.

## Features
- Basic arithmetic operations
- Scientific functions (sin, cos, tan, log, sqrt, power)
- Responsive design
- Real-time calculation display
- Error handling for invalid inputs
- Simple and modern UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was developed to improve JavaScript logic-building skills and create a practical mathematical application with an interactive user experience.

## Future Improvements
- Dark mode support
- Calculation history
- Keyboard shortcuts
- Graph plotting feature
